# Retail Sales Performance Analysis

End-to-End Data Analytics Project using Python, SQL, and Power BI.

## Business Problem

Retail businesses generate large amounts of sales data but struggle to convert it into actionable insights.  
This project analyzes retail sales data to identify key trends in revenue, profitability, customer behavior, and product performance.

## Dataset

The dataset contains three tables:

Customers – 500 records  
Products – 100 records  
Orders – 10,500 records

## Tools Used

Python (Pandas, NumPy)  
PostgreSQL (SQL Analysis)  
Power BI (Dashboard Visualization)

## Key Insights

• Electronics generates the highest revenue.  
• West region produces the highest profit.  
• High discounts reduce profit margins in some categories.  
• A small group of customers contributes a large portion of total revenue.

## Dashboard

The Power BI dashboard visualizes:

• Revenue trends over time  
• Category performance  
• Regional profitability  
• Top products  
• Customer segmentation

## Business Recommendations

• Optimize discount strategies for electronics  
• Focus marketing on high-value regions  
• Improve sales performance in weaker regions  
• Build loyalty programs for high-value customers
